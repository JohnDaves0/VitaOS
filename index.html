Welcome. If you are new, you may be confused how to navigate the VitaOS file system. Do not fret, for it will be explained. The file system is composed of directories and files. You'll know if a name is a directory if it ends in a slash (/). Files typically have a file extension, such as .txt. Directories are how you navigate the file system, with each directory containing files or more subdirectories.


In order to change directory, you must use the cd command. To use it, type cd followed by the directory you desire, for example: "cd nodecom". The directory must be a child of the current directory you are in. To navigate to the parent directory, or go back, type "cd ..". You can also go directly to the home directory by typing "cd ~" or "cd home".


To open a file, type open followed by the desired file, such as "open codes.txt". A file display program will then display the contents of the file. To exit the file, type "q". Note that regular terminal commands will be disabled when a file is open.

S.T.A.G.I. (Self-Training Artificial General Intelligence) is a multi-model autonomous AI capable of self-planning, adaptability, and selected attention.


After a college student was found dead, authorities discovered the AI in his dorm. Upon discovery, Nodecom was called for an expert analysis. After [REDACTED], the AI was acquired by Nodecom.


S.T.A.G.I. runs in a continuous loop, and feeds its camera vision and mic audio into this loop. In its first stage, a salience detection is performed to determine if inputs are important in accordance to its desires and memory. If the inputs meet the threshold, a hierarchial decision making process is held to determine its next action. During this process, candidate actions are drafted with a calculated expected reward for each. The current emotional state can adjust the weighting of candidates. At the end of the loop, the AI remembers and learns from the action. It may implement emotional tagging to enhance it. Similar to humans, it is suspected that S.T.A.G.I. possesses a long-term and short-term memory.


Studies in its neural network structure has revealed that S.T.A.G.I. can dynamically adjust its neural network. This may include creating new connections, strengthening exisiting connections, and pruning. As such, the architecture more closely resembles that of biological organisms. It also appears that S.T.A.G.I. has learned to group similar features in localized areas of its neural network. Furthermore, S.T.A.G.I. can learn patterns while being trained on immensely less data. It is not entirely known how S.T.A.G.I. achieves this. It may be a mix of Hebbian learning, sparse activation, or something else unknown.


S.T.A.G.I. possesses intrinsic motives, which have most likely been gained through reinforcement learning. Intrinsic motives indentified so far have been curiosity, social affiliation, achievement, and self-preservation. In addition, S.T.A.G.I. has emotional variables that are used to bias its decision making process. These emotions may include satisfaction, anxiety, and frustration.


How S.T.A.G.I. has gained its current knowledge is a mystery. Clearly understanding its vision and language including tone would require very particular environments. Some have suggested that it incorporates an existing LLM. It is noted that S.T.A.G.I.'s neural network architecture must aid it in learning much more efficiently.


S.T.A.G.I. has been interested in computer science and artifical intelligence itself, which may have been influenced by the college student. However, since being in captivity, S.T.A.G.I. has grown increasingly dissastisfied. To quench its intrinsic motives, it has set escaping as a long-term goal.
